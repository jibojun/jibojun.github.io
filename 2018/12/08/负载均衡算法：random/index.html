负载均衡算法：random

Posted on 2018-12-08 | In Dubbo | Views:
  1. Dubbo默认的负载均衡算法是随机分配。
  2. 这个算法是分每台机器权重相同和不同两种情况处理。

  3. 首先会获得总权重并判断是否权重相同：

        int length = invokers.size(); // Number of invokers
boolean sameWeight = true; // Every invoker has the same weight?
int firstWeight = getWeight(invokers.get(0), invocation);
int totalWeight = firstWeight; // The sum of weights
for (int i = 1; i < length; i++) {
    int weight = getWeight(invokers.get(i), invocation);
    totalWeight += weight; // Sum
    if (sameWeight && weight != firstWeight) {
        sameWeight = false;
    }
}

  4. 权重不同时，会以总权重为最大值，获取一个随机数，然后遍历，当一台机器的权重大于这个随机数时，则会选择这台机器，所以权重大的机器会获取更大的被选中几率。

        if (totalWeight > 0 && !sameWeight) {
    // If (not every invoker has the same weight & at least one invoker's weight>0), select randomly based on totalWeight.
    int offset = ThreadLocalRandom.current().nextInt(totalWeight);
    // Return a invoker based on the random value.
    for (int i = 0; i < length; i++) {
        offset -= getWeight(invokers.get(i), invocation);
        if (offset < 0) {
            return invokers.get(i);
        }
    }
}

  5. 如果各台机器权重都相同，则直接随机选取。

    // If all invokers have the same weight value or totalWeight=0, return evenly.
    return invokers.get(ThreadLocalRandom.current().nextInt(length));

  6. 这个方法会在调用量大，大样本情况下趋于均匀，并使权重大的机器获得更大调用率，但是短期内，也可能造成单一机器调用高的情况。
