switch-case的使用注意点

Posted on 2018-04-26 | Edited on 2018-05-27 | In JavaCore | Views:

注意点：switch中case语句是从满足条件的确定行向后执行，break并不是为了节省时间跳出，而是为了保证命中条件后就退出，否则在命中条件后，会继续执行后面所有case的语句直到遇到break或者case执行完毕。
所以一般用来代替if-else的选择逻辑下，需要在每个case逻辑执行完后加break跳出来避免执行后面全部语句。

测试，不加break，期望的正常结果是输出1，3，5，7，因为这四个case各一次：

    public static void main(String[] args) {
    for(int i=0;i<10;i++){
        switch (i){
            case 1:
                System.out.println(i);
            case 3:
                System.out.println(i);
            case 5:
                System.out.println(i);
            case 7:
                System.out.println(i);
        }
    }
}

实际输出，可以看出，1命中第一个条件，后面继续执行（此时不需要满足case条件），因为都没break，所以会执行完全部条件输出4个1。以此类推，3命中第二个条件，输出3次，5命中第三个条件，输出1次，7命中最后一个条件，输出1次：

1
1
1
1
3
3
3
5
5
7

Process finished with exit code 0

正确写法：

public static void main(String[] args) {
    for(int i=0;i<10;i++){
        switch (i){
            case 1:
                System.out.println(i);
                break;
            case 3:
                System.out.println(i);
                break;
            case 5:
                System.out.println(i);
                break;
            case 7:
                System.out.println(i);
                break;
        }
    }
}
